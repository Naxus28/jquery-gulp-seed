jQuery-Gulp Seed Project

This seed project is intended to help javascript and jquery developers streamline their work. It offers:

  1. A standard directory structure for front end apps.
  2. Several essential gulp tasks already configured.
  3. A development server and a production server.
  4. Support for ECMAScript 2015.
  5. jQuery Validation plugin already implemented
  6. Many Node packages and bower components already installed.
  7. A comprehensive .gitignore file.
  8. A .eslintrc.json file with configuration that enforces best code practices.
